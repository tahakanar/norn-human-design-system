# Base UI

Bu klasör Lucide tabanlı işlevsel ikon sistemini temsil eder.

## Kullanım

Kod tarafında Lucide ikonları paket üzerinden import edilmelidir.

Bu klasöre yalnızca:

- Figma icon component seti,
- seçilmiş kontrollü SVG exportlar,
- görsel preview board

eklenebilir.

## Ana Kaynaklar

- `../02_LUCIDE_BASE_SET.json`
- `../03_SEMANTIC_ICON_MAP.json`

## Kural

Bütün Lucide kütüphanesini kopyalamak veya tek bundle içinde import etmek yerine yalnızca kullanılan ikonlar seçilmelidir.
