# Norn Human Custom Icon Briefs

Bu belge, Norn Human'a özgü sekiz başlangıç ikonunun tasarım brieflerini içerir.

Bütün ikonlar:

```text
24×24 grid
2px stroke
round cap
round join
centered stroke
1px minimum safe area
2px minimum element spacing
fill none
currentColor
```

kurallarına uygun çizilmelidir.

İlk tur yalnızca siyah-beyaz hazırlanmalıdır.

---

# 1. nh-meaning

## Rol

Kendini Tanıma ve Anlam alanını temsil eder.

## Ana Kavram

**Merkez + katman + içe doğru okuma**

## Form Yönü

- tek merkez,
- açık ve kapanmayan katmanlar,
- içeriden dışarı veya dışarıdan içeri okunabilir ritim,
- göz çağrışımı oluşturmayan yapı.

## Kaçınılacaklar

- göz,
- üçüncü göz,
- mandala,
- beyin,
- aura,
- spiral şifa sembolü.

## Test

16 px'te göz veya hedef ikonuna dönüşmemelidir.

# 10. Zorunlu Teslim

```text
01 Concept statement
02 3 siyah-beyaz alternatif
03 24px grid görünümü
04 16px test
05 20px test
06 24px test
07 32px test
08 açık ve koyu zemin
09 optik hacim karşılaştırması
10 risk notu
```

Seçilen ikon daha sonra:

```text
Brand Custom/Source SVG/
```

klasörüne kaynak dosya olarak eklenir.

Onaylı exportlar:

```text
Brand Custom/Exports/SVG/
Brand Custom/Exports/PNG/
```

klasörlerine eklenir.
