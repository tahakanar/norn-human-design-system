# Photography Asset Library

Bu klasöre gerçek üretimde kullanılabilecek fotoğraflar eklenir.

Dosya adı:
`PHOTO-001_descriptive-name.ext`

Her dosya için `../00_ASSET_INDEX.md` veya ana `02_ASSET_MANIFEST.md` güncellenmelidir.

Onay kontrolü:

- kaynak ve lisans,
- ticari kullanım,
- model release,
- doğal ışık ve gerçeklik,
- yaş ve yaşam çeşitliliği,
- crop uygunluğu,
- AI hissi,
- marka klişesi riski.
